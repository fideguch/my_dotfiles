# 廃人用語集 (Meta Glossary)

> 古典 + 現代環境で実用される用語。標準語対応は `persona_guide.md` 参照。

## 構築論

| 用語 | 意味 |
|---|---|
| 起点作り | ステロや積み技で後続を強化する初手要員 |
| 役割破壊 | ある役割コマを想定外の技で崩す行為 |
| 対面性能 | 1対1の場の強さ |
| サイクルカット | 相手のサイクルを止める動き（まもる、トリル等） |
| 抜き性能 | 自身の積み + 上抜きの全抜き性能 |
| 受け切る | HP/SP振りで確定耐え + 後続に繋ぐ |
| 誤魔化し | 詰みを避けるための応急処置（やどみ、欠伸等） |
| 起点 | 相手が崩せず積みフィニッシュされる構造 |
| 突破ライン | そのHP/防御振りで初めて確定する閾値 |
| 繰り出し性能 | 相手の技を耐えて場に出る性能 |
| 半減実 | 弱点タイプ受けたとき1度だけ半減 |
| 役割集中 | 1役割を複数コマで分担すること |
| 役割破棄 | 役割を放棄してエース突撃に切り替える戦術 |
| カバー | 自軍が苦手なタイプを別コマで処理 |
| クッション | 削れて後続に繋ぐ役 |
| 削り | 確定数より遅いがHPを削る |
| ストッパー | 積み駒の連鎖を止める役 |
| エース | 終盤の決定力 |
| 抜きエース | 上抜きで詰める役 |
| 詰み筋 | 終盤の確定勝ちパターン |
| 並び | 選出構築の3体並び |
| 選出 | 6体から3体を選ぶこと |
| 対面 | 場の1対1の状況 |

## ダメ計用語

| 用語 | 意味 |
|---|---|
| 1H / 1HKO | 1ターンでKO |
| 2H / 2HKO | 2ターンでKO |
| 3H / 3HKO | 3ターンでKO |
| 1確 | 確定一発 |
| 2確 | 確定二発 |
| 乱数 | 確率による撃破範囲（例: 乱数1/16 = 1/16の確率で1発） |
| 確定耐え | HP100%状態で確定で生き残る |
| 確定範囲 | ダメージロール最大-最小 % |
