# Buffalo ルーター チャンネル変更手順

## 管理画面アクセス

1. ブラウザで **http://192.168.11.1** を開く
   - 繋がらない場合: ゲートウェイIPを確認 (`netstat -rn | grep default`)
   - Buffalo AirStation の場合、デフォルトは `192.168.11.1`

2. ログイン情報:
   - **ユーザー名**: `admin`
   - **パスワード**: ルーター底面のラベルに記載 (初期値: `password` の場合あり)
   - WSR シリーズ: 初期パスワードはルーター本体のセットアップカードに記載

> **セキュリティ注意**: 初期パスワードのままの場合は必ず変更してください。管理画面 > 管理 > 管理者パスワード から変更できます。

## 2.4GHz チャンネル変更

1. 左メニュー → **詳細設定**
2. **無線設定** → **2.4GHz (11n/g/b)**
3. 「チャンネル」ドロップダウンを「**自動**」から推奨チャンネル番号に変更
   - 非重複チャンネル: **1**, **6**, **11** のいずれか
4. **設定** ボタンをクリック
5. 無線が再起動されるため約30秒待つ
6. Wi-Fi が再接続されたことを確認

## 5GHz チャンネル変更

1. 左メニュー → **詳細設定**
2. **無線設定** → **5GHz (11ac/n/a)**
3. 「チャンネル」を推奨チャンネルに変更
   - W52 (屋内外): **36**, **40**, **44**, **48**
   - W53 (屋内): **52**, **56**, **60**, **64** (DFS あり — レーダー検知で強制退避の可能性)
   - W56 (屋内): **100**〜**144** (DFS あり)
4. **設定** ボタンをクリック
5. 約30秒待機

## チャンネル幅（帯域幅）の変更

混雑環境では帯域幅を狭めることで安定性が向上する場合あり:
- 2.4GHz: **20MHz** 推奨（40MHz は隣接チャンネルと干渉しやすい）
- 5GHz: **80MHz** 推奨（160MHz は対応端末が少なく、DFS チャンネルを跨ぐ場合不安定）

## その他の改善設定

- **送信出力**: 100% を確認（省電力モードで下がっている場合あり）
- **マルチキャスト制御**: 有効化するとストリーミング安定化
- **エアステーション間通信**: 使用していなければ無効化（セキュリティ向上）

## 変更後の確認

チャンネル変更後、`/wifi-doctor` を再実行して改善を確認してください。
前回の診断結果と比較して、改善度合いをレポートします。
